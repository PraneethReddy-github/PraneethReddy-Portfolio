## Projects
 
### CloudShare: Passwordless Cloud Storage (IEEE Published)
A passwordless cloud storage architecture using Encrypted Shamir Secret Sharing (ESSS). Instead of storing a single encrypted file, CloudShare cryptographically splits data into multiple shares and distributes them across a decentralized node network. A minimum threshold of shares must be retrieved to reconstruct the file — mathematically ensuring that compromising one node gives an attacker zero information.
- IEEE: https://ieeexplore.ieee.org/document/11118777/
### Pothole Detection and Repair System (Indian Patent, 2025)
An automated road maintenance system using high-resolution image acquisition and advanced computer vision to detect and classify potholes in real-time. Once identified, it logs GPS coordinates and triggers automated filler material dispensing for on-the-spot repairs.
 
### Cloud-Based Real-Time Anomaly Detection for Network Traffic (IEEE Published)
Uses Apache Spark's distributed computing alongside advanced ML models to analyze massive streams of network packets on the fly. Detects DDoS attacks, unauthorized access attempts, and other anomalies with high precision and low latency.
- IEEE: https://ieeexplore.ieee.org/document/11118829
### Real-Time System Resource Monitoring Dashboard
Built a system monitoring dashboard to track CPU, memory, and network utilization across Linux systems using Python, Shell scripting, and React.
 
### Road Safety & Accident Prevention Speed Zones
Designed a speed control and accident prevention system using Arduino Uno, RF transmitters/receivers, ultrasonic sensors, and embedded C.
 
### Secure Multi Client-Server Communication using SSL
Secure client-server communication using SSL/TLS with hybrid RSA-AES encryption, secure key exchange, digital signatures, and encrypted message transmission.
 
### Genetic Algorithm for Intelligent Vehicle Routing
